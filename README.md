A dungeon and dragons dice bot designed for Discord.
Can do the following:
- Roll dice in multiple quantities of multiple sides
- Allow for players to import character sheets saved as JSON files
- Allow for players to level up their characters within the bot
- Allow for players to cast spells and have their damage values calculated
- Allow for players to use shortcuts for saving throws
- Allow players to store multiple character sheets with custom icons
- Allow players to roll and track death saves
- Allow players to manage HP including via short and long rests that are automatically calculated
- Allow players to multiclass
- Allow players to save their spells as a loadout which can be switched
- Allows players to keep track of and manage spell slots
- Full initiative tracking for ease of combat
